# Kinetik Templates

First-party templates live here.

Templates are complete project-shaped acceptance targets. They are different
from `examples/`, which remains for small illustrative samples that are useful
to read but are not product acceptance gates.

Before adding or changing a template, follow:

- `docs/architecture/template-projects.md`
- `docs/architecture/project-organization.md`
- `docs/quality/testing-strategy.md`

Each template must include `README.md` and `VERIFY.md` before content can be
treated as complete.
